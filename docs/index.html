회원

가입·로그인·마이페이지, 아이디/닉네임 중복확인(Ajax), 아이디·비밀번호 찾기와 회원 탈퇴까지.

Ajax 중복확인

게시판

공지·자유·이벤트 카테고리, 글 CRUD(소프트 딜리트), 댓글, 검색·페이지네이션, 신고와 이미지 업로드.

소프트 딜리트

메뉴

DB 기반 동적 렌더링, HOT/ICE 가격 차등과 사이즈 추가금. 결제 시 DB 가격을 재조회해 위변조를 차단합니다.

서버측 가격검증

멤버십

일반·실버·골드·VIP 4단계. 방문 횟수를 조인해 등급을 자동 재계산하고 다음 등급까지 진행도를 보여줍니다.

등급 자동 재계산

장바구니

세션이 아닌 cart_t DB에 영구 저장. 우측 슬라이드 패널에서 옵션별 수량을 누적·관리합니다.

DB 영구 저장

주문 · 결제

TossPayments 실연동에 금액 위변조 검증. 시간 기반 3단계 차등 환불(3분 내 전액 / 3–10분 50% / 초과 불가).

시간기반 차등환불

AI 챗봇 ‘아메리’

Groq LLaMA 3.3 70B

Architecture

히스토리를 보존한 모노레포

원래 분리돼 있던 두 저장소를 git subtree merge 방식으로 하나로 통합했습니다. 각 프로젝트의 과거 커밋이 그대로 남아 있어 git log로 양쪽 이력을 모두 확인할 수 있습니다.

두 앱은 서로 다른 패키지·빌드 설정을 가진 독립 Spring Boot 프로젝트로, 각자 빌드·실행됩니다.

Quick Start

직접 내려 마셔보기

두 프로젝트는 각각 독립적으로 빌드·실행합니다. 자세한 내용은 각 디렉터리의 README를 참고하세요.

Front of House

ROWNCafe (고객용)

# 저장소 클론 후 cd 2026-CafeRown # .env.example → .env 복사 후 DB·Groq 키 입력 # Docker 권장 docker compose up -d --build # → http://localhost:8080/

Back of House

BREW CRM (백오피스)

cd 2026-CafeRown2 # schema.sql 로 DB 생성 후 # application.properties 수정 mvn spring-boot:run # → /brew-crm/customer/dashboard
